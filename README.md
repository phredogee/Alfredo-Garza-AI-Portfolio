## 🚀 Featured Projects

### 🔹 [NewsBot Intelligence System 2.0](https://github.com/phredogee/ITAI2373-NewsBot-Final) — NLP Final Project

An end-to-end NLP pipeline that transforms raw BBC news articles into structured insights. Achieved **98.88% classification accuracy** using Logistic Regression with TF-IDF vectorization. Features include sentiment analysis (VADER + Transformers), Named Entity Recognition (spaCy + HuggingFace), topic modeling (LDA), and a prototype multilingual pipeline.

**Tech:** Python · spaCy · HuggingFace · scikit-learn · NLTK · TF-IDF · LDA

---

### 🔹 [NewsBot Intelligence System — Midterm](https://github.com/phredogee/ITAI2373-NewsBot-Midterm) — NLP Midterm Project

A modular NLP pipeline for BBC news article classification using TF-IDF feature extraction and three ML models (Naive Bayes, SVM, Logistic Regression). Includes preprocessing, POS tagging, dependency parsing, sentiment classification, and NER with full evaluation reports and confusion matrices.

**Tech:** Python · NLTK · spaCy · scikit-learn · pandas · matplotlib

---

### 🔹 [GridIron GPT](https://github.com/phredogee/gridiron_gpt) — LLM Application Project

An AI-powered fantasy football advisor using semantic search and LLMs. Ingests live NFL data (injuries, rankings, matchups), indexes it with FAISS vector embeddings, and generates natural language start/sit recommendations via OpenAI or Mistral. Includes a decision tree classifier, Slack notifications, and a CLI interface.

**Tech:** Python · FAISS · HuggingFace · OpenAI API · Mistral · scikit-learn · Slack SDK

---

### 🔹 [ML Midterm — Adult Income EDA](https://github.com/phredogee/ml-midterm-eda) — Machine Learning Midterm

Exploratory data analysis and predictive modeling on the UCI Adult Income dataset. Covers data cleaning, feature engineering, and supervised classification to predict income levels.

**Tech:** Python · pandas · scikit-learn · Jupyter Notebook
